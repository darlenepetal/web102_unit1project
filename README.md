## Notes

- Couldn't figure out how to handle longer recipe names such that the text would stack neatly, which I would like to figure out how to do
- Justifying/aligning elements was/is quite confusing, as I'm not sure what the differences are between "justify-content," "align-items," "place-items," etc.
- I found that "justify-content: space-evenly;" is much better looking than "justify-content: space-between;," which was used in the examplar so I went with the former
- There was also a bug with the examplar where the top of a Card was not rounded, but the bottom was. This was due to the img tag inside of a card not having a border radius, which I added in only the top corners of the img as to create rounded corners around the entire card
